# Apps

## Overview

This section covers security recommendations for Google Workspace applications. These controls help ensure that Google Workspace apps are configured securely and data is protected appropriately.

## Recommendations

### 3.1 Google Workspace Applications

#### 3.1.1 Calendar
Controls in this section focus on calendar sharing and access:
- External sharing settings for primary calendars
- Internal sharing settings for primary calendars
- External invitation warnings

#### 3.1.2 Drive and Docs
Controls in this section focus on file sharing and access:
- External sharing settings
- Internal sharing settings
- File access controls
- Sharing permissions

#### 3.1.3 Gmail
Controls in this section focus on email security:
- DKIM configuration
- SPF records
- DMARC configuration
- Email security settings

#### 3.1.4 Google Chat
Controls in this section focus on chat security:
- External access settings
- File sharing controls
- Chat history settings

#### 3.1.5 Google Meet
Controls in this section focus on meeting security:
- External participant settings
- Recording controls
- Meeting access settings

#### 3.1.6 Groups for Business
Controls in this section focus on group security:
- External access settings
- Group creation controls
- Group membership settings

#### 3.1.7 Sites
Controls in this section focus on site security:
- External sharing settings
- Site creation controls
- Site visibility settings

## Implementation

Each control in this section is implemented using SQL queries against the Steampipe Google Workspace plugin tables. The queries check various aspects of application configuration and return one of the following status values:

- `ok` - The control check passed
- `alarm` - The control check failed
- `info` - The control check found something that should be reviewed
- `skip` - The control check was skipped

## Best Practices

1. **Email Security**
   - Enable DKIM for all domains
   - Configure SPF records properly
   - Implement DMARC policies
   - Monitor email security reports

2. **File Sharing**
   - Set appropriate default sharing settings
   - Regularly review file sharing policies
   - Monitor external sharing activity
   - Implement DLP policies

3. **Communication Tools**
   - Configure appropriate meeting settings
   - Control external participant access
   - Enable chat history for compliance
   - Monitor communication patterns

4. **Access Control**
   - Implement least privilege access
   - Regular review of sharing settings
   - Monitor external collaborations
   - Audit access patterns
